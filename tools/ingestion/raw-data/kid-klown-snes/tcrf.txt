It’s a game that has a clown getting hurt but you don’t wanna get hurt so you can get to a BOMB IN TIME. Also you need to get all the cards on the first run down to get a 50/50 chance to beat the game and there’s a hard mode and you need to get all cards in that mode on the first run down to get another 50/50 chance to beat the game and see the credits.

Level Select

At the title screen, hold  L+R  on both controllers and one of the below buttons on controller 2, then press the same button on controller 1 to go to the corresponding stage.

Sound Test

Beat the game, then when the Kemco logo appears during the ending, press and hold  L+R  until the title screen reappears. A new option called "???" will be added to the title screen. Select it to access the sound test.

The Japanese version has actual titles for the songs, but the International versions replace them simply with a number.

(Source:  urawaza.in )
