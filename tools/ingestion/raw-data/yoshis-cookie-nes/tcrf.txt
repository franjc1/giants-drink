Yoshi's Cookie

is a puzzle game in which you match rows and columns of cookies in different shapes.

In 2013, the game was pulled from the Wii Shop Channel for unknown reasons.

Development Info

Debug Menu

Game Genie  code

GEUGXXGK

(Japan) or

GEUKNXGK

(US) re-enables the debug cheat code, which allows you to access a debug menu by pressing  A + B + Down + Right  at the title screen.

The debug menu allows you to select any game mode/round/stage/speed, listen to the sound effects, or view all mini-cutscenes and both staff sequences. To return to the title menu, select the "T" option. Press  Up  and  Down  to select an option,  A  and  B  to change option parameters, and  Start  to activate an option. A similar debug menu is also present in  the Game Boy version .

Normally, the "SND" option doesn't work due to a bug which just crashes the game. This can be fixed with Game Genie codes

AAXLSPAZ OAXLVPAA XTXLNPOE

(Japan) or

AEUUAIAZ OEUUPIAA XVUUZIOE

(US).

Mapper Remnants

The game may have been initially developed for a different mapper before being hastily ported to MMC3. (e.g. scanline IRQ and PRG-RAM states are never initialized)

An unused bankswitch routine is present in all versions (PRG offsets differ, but are always located before the reset handler in the fixed bank):

TAY
STA $8000,Y
RTS

This is corroborated by the fact that the first 256 bytes of every switchable 16 KiB PRG "bank" (the game only ever switches 2 x 8 KiB banks at a time) mapped at

$8000

is zeroed out, which may have been reserved for a bank lookup table (in order to work around bus conflicts on early discrete logic mappers).

The game also writes a constant 1 to

$6000

after switching CHR-ROM banks for each screen/game mode, despite never initializing the write-enable/protection state for any RAM mapped there.

(Source:  TakuikaNinja )

Regional Differences

Japan

US/Europe

The Japanese logo has two sets of eyes included and is in a large cloud. The international logo has no eyes (but does have a Yoshi Egg) and is in an orange box.

The clouds surrounding the logo and the "1P OR VS" section were redrawn.

The international versions updated the copyright info and added a credit for Bullet Proof Software.
