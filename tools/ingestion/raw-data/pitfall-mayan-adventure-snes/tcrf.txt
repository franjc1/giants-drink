Debug Mode

At the title screen, press  X, Select, A, Select, Y, A, X, Select  for a stage select.  L  and  R  change the stage.

While in a stage, hold  Select  for a free-movement mode. A coordinate display will also be available. Also, pause the game, then press  Select  to return back to the title screen with a stage select.

Unused Option

At the title screen, enter the "OPTIONS" menu. Change the RAM value at

$7E:$027E

to

$06  . Use  B ,  X  and  Y  to input a code. Press  Start  to accept the code. Because of wrong script condition test opcode, the only code that won't be accepted here is  BXYXYYBX . All other inputs are accepted and enables the debug mode mentioned above. Obviously, condition test here is inverted and had to test if code is exactly  BXYXYYBX . It seems that this was a planned option to enter a secret cheat code which was dropped at some point of development and replaced with button codes which uses the buttons  Select ,  X ,  Y  and  A  instead.
