Felix the Cat

is a somewhat short, fairly easy, and incredibly unique platformer for the NES. The Professor hasn't stolen Felix's magic bag this time; he's kidnapped Kitty! Go save her.

Prerelease Info

Japanese Text

Although the game wouldn't release in Japan until another 31 years, Japanese text exists for the cutscenes in the international ROM, along with code to load the appropriate characters into the PPU. Edit RAM address

$03C8

to be

$00

rather than

$80

to see this in-game. Alternatively, use one of the following Game Genie codes to see it in-game:

AEUGTZEA

(US version, affects PRG ROM address

1c2c6  ),

AEUKPZEA

(European version, affects PRG ROM address

1c2b9  ),

AEKEALEA

(unknown version).

(Source: Divingkataetheweirdo (initial find; GG code conversion), 20kdc (Code discovery), Qalle (more versions of the GG code))

Revision Differences

Felix the Cat would see a re-release in 2024 by Konami for Nintendo Switch, PlayStation 4, and PlayStation 5. There were several changes made that apply to both the English and Japanese versions. Notably this re-release would correct the aggressively dark palette the original release had.

Title Screen

Original

Re-release

Nintendo Switch

PlayStation

The ™ above the Felix the Cat logo was removed in the re-release. "Push Start Button ǃ" was changed to "Push + Button ǃ" on the Nintendo Switch release and "Push Options Button" on the PlayStation 4 and 5 release. Additionally, the copyright information was updated to remove mentions of Hudson Soft and Felix the Cat Productions, Inc. and instead replaces them with Felix the Cat™ and DreamWorks Animation LLC. "Licensed by Nintendo" would be replaced with "All Rights Reserved". The purple bar that's under Hi-Score was also removed.

Original

Re-release

Poindexter in the original is swapped out with Kitty Cat in the re-release. Additionally, Rock Bottom is no longer smoking a cigarette in the re-release.

The reason for Poindexter's replacement is due to the rights for the character being owned by Don Oriolo while the rest of the Felix the Cat characters are owned by NBCUniversal/DreamWorks.

Graphic Changes

Original

Re-release

The magic wand sprites were changed. In the original, Felix faces forward. In the re-release, Felix remains facing sideways.

Original

Re-release

The tank was also changed. The re-release has Felix lowered down and is behind glass. The tank appears more toy-like in the re-release.

Original

Re-release

The cannonball in the original is black where as in the re-release it's orange.

Original

Re-release

Poindexter is replaced with The Professor.

Original

Re-release

As was changed with the title screen Rock Bottom is no longer smoking a cigarette. The gun was also changed to be less realistic.
