The SNES port of

Operation Thunderbolt

is a Super Scope enhanced port of the arcade original with a different storyline and characters to choose from.

Development Text

Stored right at the beginning of the ROM is a version number.

SNES Operation Thunderbolt v1.00

Starting at

0x686

is some debug messages for the SNES Mouse, as with several other SNES games that has support for the SNES Mouse.

START OF MOUSE BIOS
MODIFIED FROM SHVC MOUSE BIOS Ver1.00
END OF MOUSE BIOS

Anti-Piracy

On every frame where the game's logo is on the title screen, the game checks if there is any SRAM present by writing a value at

$30626B

and comparing it with

$30626C

to ensure they are equal. If the write was successful, this will result in a mismatch that causes the game to disable all controller input, preventing any further progression past the title screen. An original cartridge does not contain any SRAM and thus the write will do nothing, but a cartridge copier would include large amounts of SRAM in order to support a wider variety of games, thus tripping this check.

(Source:  Original TCRF research )
