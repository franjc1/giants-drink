Debug Mode

Press  A (×4), X, B (×4), Y, X (×4), A, Y (×4), B  on the title screen. A sound will confirm correct entry.

Now, while on any side-scrolling level, press one of the following button combinations on Controller 2 for the desired effect:

Debug Menu

Press  L + R .

Area  - Not to be confused with "Stage", this selects the stage proper.

Stage  - Select part of the current stage.

Lives  - Select up to 99 lives.

Health  - Set and/or refill health.

Actor  - Select active player, but only from those available for the current stage.

Weapon  - Change weapon.

Stage Skip

Press  Start .

Invincibility and Player Coordinate Display

Press  A + B + X + Y + Select .

Sound Test

During the game, hold  A + B + X + Y  and press  Start  on Controller 1.

Revisional Differences

The game was rereleased by Nintendo of America in 1996 as part of their "Players Choice" series. As JVC was no longer involved in publishing duties, a few changes were made.

1992

1996

The very first screen changed "JVC" to "NINTENDO".

1992

1996

The copyright info on the main menu was also altered: "LICENSED BY NINTENDO" became "LICENSED TO NINTENDO", and "LUCASARTS ENTERTAINMENT" was changed to "LUCASFILM LTD."
