Miscellaneous

This game has an invalid SNES header of 0x53. It is incorrectly detected as FastROM, but, it's not. This error appears to have been introduced in the original cart itself.

The cart title is "CONTRA3 THE ALIEN WARS", but, this is 1 letter too long and as a result, overflows the 0x15-sized buffer. The byte incorrectly overwritten in the header at 0x7FD5 is the "S" in "ALIEN WARS".  "S" = 0x53, which is not a valid mapper, but, most tools detect the FastROM bit set, and default to LoRom.

The Japanese version is correctly reported as 0x20 which is LoRom + SlowRom.
