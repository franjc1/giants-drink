Miscellaneous

StarTropics is one of two games produced by Nintendo that has never been released in Japan, and it is the  only  NES game created by Nintendo that has not been released in Japan.

Using Game Genie code PAEAPAAA, the player can hold the START button to allow Mike to travel on any tile during the Travel Stage (name given to the map screens in the manual). If this code is entered upon the game starting up, the player can hold START to select from a list of the Dungeons in the game. If SELECT  and  START are held when the game is started, then the player can choose a Chapter to play at. If SELECT is held when the game is started, then StarTropics' ending will play.

An annotated disassembly and documentation of the music code was published with this hack:  RHDN/hacks/2229: StarTropics (Music Fix)
