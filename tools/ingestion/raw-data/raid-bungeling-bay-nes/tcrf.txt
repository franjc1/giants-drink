Title Screen

Hidden Message

The Japanese version contains a hidden developer's message, triggered by pressing  Up + B + Select  while the demo gameplay is shown. The string "BY KIKU" will appear at the bottom of the screen.

"KIKU" is a pseudonym for Kikuta Masaaki (菊田昌昭), one of Hudson's programmers. Most games programmed by Masaaki have secrets or hidden messages, including

Nuts & Milk  ,

Binary Land  , and

Challenger  .

Although the US version contains the string and the code, it uses a completely different title screen display routine, and uses a different title screen text block from the CHR ROM data, so a grey screen is shown instead of the title screen.

Microphone Support

In the Japanese version's 2-player mode, use the microphone in the second controller and a "Mic" icon will appear under the score. If you repeatedly use the microphone, the icon will turn red. Keep using it and eventually up to six fighter jets will appear and attack you. While this can't be triggered in the US release due to the second controller not having a microphone, the functionality is still present and has not been dummied out.
