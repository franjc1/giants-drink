Prerelease Info

Stage Select

At the title menu, select the "HI SCORE" option, hold  Right  and press  A  or  B  on the first controller. Instead of the "RECREATION SYSTEM V1.00" screen, the "DARKMAN CONTROL SCREEN" login appears. In this screen, enter "DERM" as the password, the last letter can be anything. Now you can select any of the sublevels or boss shooting events as well as ending sequence.

Hidden Text

Like many other games published by Ocean, there is a hidden version and copyright string for the music driver:

MUSIC DRIVER V1.0 (C)JONATHAN DUNN & OCEAN SOFTWARE

Unused Graphics

A small Darkman head, stored near the item graphics. An extra life, maybe?

A Nintendo logo, stored with the rest of the graphics for the copyright screen.
