Saving and Loading

In the Japanese Famicom version, players can save and load created courses with the Famicom Data Recorder plugged into the  Family BASIC  keyboard, which uses audio cassettes to do so. However, since the device was never released outside of Japan, neither of these work on most other releases. Both options still have their programming fully intact and function as intended if this device were present.

Excitebike 64

The loading and saving system was added to the version included in

Excitebike 64  , allowing one custom track to be saved and loaded on the N64 cart.  &#91;1&#93;

Animal Crossing

NES

Animal Crossing

The

Animal Crossing

version removes the Load and Save options, but the game's NES emulator supports one save per NES game per Memory Card, which occurs automatically when pressing  L + R + Z  to quit an NES game. This save data is automatically loaded when the game is played again, replicating the ability to save one custom track.

3D Classics Excitebike

The save and load functions are present in the  3D Classics  version on the Nintendo 3DS.

Nintendo Classics - Nintendo Entertainment System

The Nintendo Classics release for the Nintendo Switch Online service implements the save and load functions via SRAM.
