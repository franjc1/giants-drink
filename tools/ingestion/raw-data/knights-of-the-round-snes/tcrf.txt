Hidden Option Mode

A secondary Option Mode screen remains in the game, though the code that activates it (located at SNES address

$80C9A1

in the US version) was disabled before release. It can be reenabled with the following  Pro Action Replay  (PAR) codes:

US

80C62E00
80C62FFA
80FA0020
80FA01A1
80FA02C9
80FA0320
80FA04E1
80FA05C9
80FA0660

Japan

80C5C720
80C5C8F9
80F92020
80F9213A
80F922C9
80F92320
80F9247A
80F925C9
80F92660

Once the codes are enabled, highlight "EXIT" on the options screen and press  Select  on both controllers simultaneously.

Unfortunately, the "PLAYER DEATH" option is not functional, as the RAM address that holds the flag (  $7E0200  ) is never checked during gameplay. Either the address changed during development, or the code was removed.

(Source: gedowski, JLukas, Rachel Mae)

Debug Display

The Pro Action Replay (PAR) code

81F954EA

will add this hexadecimal display to the screen during gameplay, which contains information about the current screen scroll position.

(Source: Original TCRF research)
