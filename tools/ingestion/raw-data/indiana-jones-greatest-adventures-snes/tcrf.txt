Indiana Jones' Greatest Adventures

is a  Super Star Wars -like platformer chronicling Indy's first three on-screen adventures, commonly considered to be his greatest.

Button Codes

During gameplay, pause the game, enter any one of these three codes on controller 1, then unpause. (Only one code can be entered per pause.)

Code

Effect

A, Y, Left, Up, Down, Right, X, Y

Enables level skip (pause, then press  B )

L, R, Down, Up, Y, X, Select

Adds a level select feature to the options menu

Right, Left, Right, Left, Up, Down, Y, A

Takes away all of the player's lives, resulting in a game over

Note that resetting the game causes these codes to stop working, so the only way to use the level select feature normally is to get a game over, either by losing all lives or by using the third code to get a game over immediately.

There is also a flag enabling invincibility, but no button code to enable it. The Pro Action Replay (PAR) code

7E1E9A0D

will enable invincibility in addition to the level skip and level select.

(Source: Original TCRF research)
