Debug Passwords

169-963-491 : Enables a cheat mode. During the game press  Select + Start  to skip the current level. Hold  A , then press  Select + Start  to restore health, fuel and ammo for all weapons.

000-000-047 : Enables cutscene debug mode. Each frame waits for you to press  A  before continuing.
