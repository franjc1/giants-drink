Porky Pig's Haunted Holiday

is another mascot platformer on the SNES. Its most innovative feature is its random environment system, where levels will change their appearance -- for example, the first level can be set in spring, autumn, winter, or Christmas.

Alternative Options Screen and Joe Booth's C.M.S. Editor

To do:

Figure out what CMS Global 4-6 and CMS Active Mode do.

With how opaque these flags are, there's probably more effects than I'm not aware of.

On the license text screen, wait until just before it fades out and quickly press  Down, Up, Left, Left, A, Right, Down .  This sets

7E1E7B

to

01  .

After that, the Alternative Options Screen appears. Selecting one of the items here will play the respective sequence, then return to the Alternative Options Screen when done.

To access Joe Booth's C.M.S. Editor (named for the game's programmer), press  Select  during gameplay. The editor allows you to edit various global variables, most of which are related to the random environment system:

CMS Flag

Effect

Room No.

Self-explanatory. A full list of what numbers correspond to each room can be found in the  notes page .

Sunsoft Intro

Selects what randomised variant of the Sunsoft logo animation is used. Annoyingly, the only way to see the effects of this is to game over, as the game doesn't preserve any memory after a soft reset.

CMS Global 1

Changes the general appearance of The Haunted Woods as well as the palette of the swimming pool rooms in the castle. Setting it to

$01

will enable rain in the intro room of The Abandoned Mines, and setting it to

$03

will enable the Halloween theme in Atlantis.

CMS Global 2

Changes various background decorations in the game. Strangely enough, setting it to

$01

enables various graphics not normally seen in the game! (see below)

CMS Global 3

Also affects the game's background decorations.

CMS Global 4-6

Currently unknown.

CMS Global 7

Affects how heavy rain/fog is in areas that support it. Ranges from

$01

to

$03  .

CMS Global 8

Enables snow in level themes that allow it when set to

$01  . It also replaces the lamps in The Abandoned Mines with light bulbs.

(Source:  Original TCRF research )

Unused Items

Bonus items that never appear in the game, leftover from an  earlier build . Up to 3 of them can be obtained at once, and were used to access the Porkathlon minigames in a (coincidentally) similar fashion to the animal tokens in  Donkey Kong Country , however this functionality has since been disabled. Use the  Game Genie  code  D7DC-BE78  to give yourself these items.

Interestingly, each level has their own versions of these bonus items!

(Source:  Original TCRF research )

Unused Graphics

Porky

Unfinished frames that were meant to be part of Porky's ducking animation. They are slightly different from the surrounding finished frames and may have been left out to speed up the animation.

Intro Objects

Haunted Woods (Spring)

A big-eyed frog. When CMS Global 2 is set to

$01  , it will appear and jump into the pond by the mailbox.

Haunted Woods (Autumn)

Some kind of cuckoo bird. When CMS Global 2 is set to

$01  , it will spring out of the hole above the window on Porky's house.

Dry Gulch Town

An odd looking bird. When CMS Global 2 is set to

$01  , it will fall from the sky and leave a bird-shaped hole on impact with the ground.

Atlantis

You'd  think  this would be an enemy intended for the Mines level, as it's littered with TNT boxes. Instead this appears on top of the Atlantis billboard when CMS Global 2 is set to

$01  , where it crawls forward for a little bit before exploding.

The Alps

Looney Tunes character heads, probably meant for the large Alps sign.

Miscellaneous

HUD Font

The full set of graphics for the numbers in the HUD also includes the letters A-F, presumably for some sort of debugging display.

Demo Graphic

Same style as the PAUSED graphic. Probably used for either gameplay demos or a convention version of the game.

Porkathlon Heads

These graphics of Porky and Daffy's heads are loaded in the Options menu, also intended for the scrapped Porkathlon mode. The Porky graphic is also depicted in a different art style from  earlier in development .

(Source:  Original TCRF research )

Unused Text

File Text

Text left in the ROM from the compiling process. Psy-Q is a development kit that was used for various systems.

E:\PSYQ\PORKY2\TEXT.658
E:\PSYQ\PORKY2\CMS.658
E:\PSYQ\PORKY2\SCROLL.658

Sound Copyright Text

Copyright text for an audio driver that's never displayed in the game.

SLICK/Audio
1.01 Copyright(C)1994 Bitmasters, Inc.

(Source:  Original TCRF research )

Build Date

At

0x2CF7

is a build date, which is displayed in-game at the bottom of the C.M.S. Editor screen.

BETA.01 AT 17:48 ON 22/8/1995
