Stinger

is a 1983 horizontally  scrolling shooter

video game  released by  Seibu Denshi  for  arcades .

&#91; 3 &#93;

It was the first game released by this company. &#91;

citation needed

&#93;  Despite the horizontal scrolling, the game has a vertically oriented screen. On July 24, 2025, the game was ported to  Nintendo Switch  and  PlayStation 4  as part of the  Arcade Archives  series.

&#91; 4 &#93;

Gameplay

[

edit

]

The game consists of traveling  around a space fortress meanwhile shooting aliens and spaceships. An object called "Bongo" could be used as a defense method. This object will shoot itself to the enemies for a few seconds.

Reception

[

edit

]

In Japan,  Game Machine  listed  Stinger  on their November 1, 1983 issue as being the sixth most-successful new table arcade unit of the month.
