Gyromite

was the first and more common of the two Robot Series games (designed to be used with R.O.B., the other being

Stack-Up  ), mostly because  Gyromite  was packed with NES Deluxe Sets.

Bizarrely, like  Stack-Up , early  Gyromite  cartridges merely contained a copy of the Famicom's  Robot Gyro  with an adapter so it could work on the NES.

Of note is that  Gyromite  can be played perfectly well without R.O.B. since the gates are moved by using Controller 2, making it a more fast-paced co-op game (along with being an almost-out-of-the-gate hint that R.O.B. was just a gimmick).
