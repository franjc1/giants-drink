Hidden Screen

FDS

NES

In the NES version, use Game Genie code

NNTYEN

to show this screen on power-on. In the FDS version, you will have to overwrite

$DFE0

in RAM with the value

$FF  , then soft reset.

The "secret tricks" referenced are:

Holding  A+B  on both controllers and then pressing  Start  on the title screen will remove the goalies.

Holding  A+B  on both controllers and then pressing  Start  on the team selection screen will prevent the puck from being slowed down by friction.

Version Differences

Ice Hockey  was first released in Japan on January 21, 1988 for the Famicom Disk System. The game would later be released for the NES a few months later in the US and Europe.

Note that  Ice Hockey  was also released for the  FamicomBox  which uses the NES version. The FDS version is playable in

Animal Crossing: New Horizons

in all regions.

Teams

FDS

NES

Japan was replaced with Sweden. The first two teams in the lineup were also swapped so USA appears first.

FDS

NES

Despite this, the player types for the CPU teams (pictured on the right-hand side of the screen) weren't edited to reflect this change.

Music

FDS

NES

The main game music was completely redone. Interestingly, the  Ice Hockey  microgame in

WarioWare: Twisted!

uses the FDS theme in all regions.

FDS

NES

The jingle that plays after a goal is scored was changed to the traditional victory theme usually heard in real-life ice hockey matches. Interestingly, the FDS jingle can still be heard in

NES Remix 2

as part of the main theme.
