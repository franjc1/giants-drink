Version Differences

Unused Graphics

When you defeat an opponent, they turn into a frame of their heads. However, each one has an unused extra frame.

(Source:  Random Hoo Haas )

Unused Music

Using the Pro Action Replay (PAR) code

7E010013

will play an unused track that could be an early version of Wario's theme or a test track.

(Source:  nensondubois )
