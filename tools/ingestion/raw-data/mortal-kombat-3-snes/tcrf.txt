Unused/Hidden Text

Build Text

A build string is stored at

0x00F77A

in the US version and

0x00F78B

in the European version. To see it, hold  L + R  when the game starts.

U.S.

SNES VERSION: WED=AUG=02==11:32 AM

Europe

SNES VERSION: FRI=AUG=11==11:51 AM

Slow/Fast

The words "SLOW" and "FAST" are stored with the rest of the option menu text at

0x02FB3C  . These aren't used anywhere. There's a timer option in  the Genesis port  that uses these, but it's not present in this version.

Sound Driver Message

Present at

0x0E8075  :

SNDDRVR
MK3  HI MOM

0x0E987D  :

ret
SetNextEffectPan

mov NextExStereoMod,

Other Strings

MK3 ARCADE KODE

No idea what this means.

NINTENDO POWER EXCLUSIVE
SHADOW FIGHTING ENABLED
ALL BACKGROUND BLAST
ALL PROJECTILES BOUNCE BACK

These strings appear in the list of Kombat Kodes, so maybe some more KKs were planned for the SNES version.

According to an issue of Nintendo Power, the Nintendo Power Exclusive code would have stopped the timer and have both fighters take minimal damage, with the intent being like a sort of "Practice Mode".

THE VORTEX

This is apparently the name of the hidden stage (where you fight Smoke); it never appears in the game.
