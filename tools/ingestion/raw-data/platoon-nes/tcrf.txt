Notes

Debug Leftovers

Game Genie  codes

ZONAPEIZ XENAZEOS

will partially re-enable a stage edit mode for the "Jungle Confrontation" levels. Press  Start  during this stage to enter the edit mode instead of pause. On Controller 2, use the  D-Pad  to move the obstacles,  A  to change the obstacle type, and  B  to change the level.

Unfortunately, due to using the same RAM variables in the debug and in-game code, the level change option will likely crash the edit mode. To fix it will require some more hacking (see  the Notes page ). In any case, this edit mode can't actually edit anything or place obstacles in the level.

Secret Names

Entering  SUZUKIKUN  or  HIROKUN  in the high score name entry screen will let you see the ending sequence, while entering  KOMADA  will let you access a music test screen.

"SUZUKIKUN" and "KOMADA" refer to Keiichi Suzuki and Yoshiaki Komada (Don Gavachyo), both of whom are listed in the game's credits under Programming. "HIROKUN" refers to Hiroaki Higashiya, who is listed in the credits as Program Director.

(Source: Cah4e3)

Unused Music

An unused track titled "Intermission" can be found in the NSF. This song was used on the start screen of the tunnel system level in the Commodore 64 version of the game, but it's silent in the NES version.
