Flipull  for the Famicom is a port of the  arcade game  of the same name.

Stage Select

Press  Down, Left, Right, Up, A, Right, Down, B, Up, Left, Start  on the title screen.

Use  A  and  B  to scroll and  Start  to start on the selected level.
