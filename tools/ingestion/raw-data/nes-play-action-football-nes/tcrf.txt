An average football game with a Nester cameo.

Disabled Debug Feature

The Game Genie code

AENNONNY

re-enables a simple debug feature, which disables all collision detection from player 1.

(Source: CaH4e3)

Virtual Console Version

For the Virtual Console version, references to the National Football League Players Association were removed, and all players were given fake names, though their stats, numbers, and face graphics remain the same.
