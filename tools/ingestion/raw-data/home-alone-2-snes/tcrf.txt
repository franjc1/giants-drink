Unused Game Over Screen

Pro Action Replay  (PAR) code

7E045A03

will load an unused (and clearly incomplete) game over screen (the final game simply displays "Game Over" over the screen when Kevin loses all of his lives). The only way to exit this screen is to reset the game. This screen  was  used in the  Game Boy version  though.
