Debug Menu

Using the  Pro Action Replay  (PAR) code

0182EA01

(US) /

0182E501

(JP) /

01849F01

(EU), then holding  Select  and pressing  Start  at the title screen, will display a debug menu that includes the following options:

STAGE SELECT : Allows you to choose a stage to play.

OBJ TEST : A controllable character animation test.

TEST : View the ending.

SE TEST : Listen to various sound effects.

BGM TEST : Listen to various music tracks.

Debug Menu Controller Input:

Up/Down : Navigate menu.

Left/Right : Increase or decrease value of selected item.

Start : Activate selection.

(Discovery: Abystus)

(Source:  Original TCRF research )

0x1C1

NATSUME 1993

Sound Test

A standalone sound test can be found by holding  L  and  R  at the title screen and pressing  Start . Press  Y  to play the selected song, or press  Start  to return to the title screen.

(Source: GameFAQs)

Level Select

A standalone level select is also available, by holding  X  and  Y  at the title screen and pressing  A, A, A, A, B, B, B, B, A, B, A, B, A, B, A, B .

Additionally, entering the code again on the level select screen allows you to select the area of the level to start in as well.

Hit Effects

Japan

International

In the Japanese version, all hit effects are shown as green blood. The international releases changes this to generic hit effects due to Nintendo of America's stricter content regulations at the time.

Female Ninjas

The Japanese version features female ninja enemies known as Kunoichi. They were removed in the international versions, with all of their appearances replaced by Monkies (the small ninjas with claws).

Anti-Piracy

At various points (such as every frame when the "PRESS START" prompt appears on the title screen and during screen transitions), the game will attempt to write to SRAM at

$700000  , which doesn't exist on a genuine cartridge. If the writes succeed, various things happen to make the game more difficult:

The level select code mentioned above doesn't work.

Health pickups don't actually do anything.

Health doesn't automatically refill to max after completing each stage.

Dying immediately sends you back to the intro instead of allowing you to continue.

This check is only implemented in the American and European versions of the game; the Japanese version of this game does not have this check.
