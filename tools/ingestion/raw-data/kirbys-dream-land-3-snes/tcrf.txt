Kirby's Dream Land 3

is the third game in the "Dream Land" series, and the only one to see a release on the SNES. In addition to Kirby's usual arsenal of powers, he also has the aid of Gooey, a strange sentient blob, and six animal friends, who can alter his powers drastically! Also notable for being the last first-party SNES game released in North America.

Debug Mode

This needs some investigation.

Discuss ideas and findings on the  talk page .

Change ROM address

$387E

to

01

(code:

00B87E01  ) to enable a rather extensive debug mode, which can be accessed by pressing Select on the title screen. If you press the L and R buttons at the same time on the Title Screen the ROM build date will be shown for as long as you maintain these buttons pressed. The build date does not differ between the Japan and USA versions, and is located in

0x10000

of the ROM.

There are seven pages worth of options, which can be cycled through with the L and R buttons. Pressing A will exit the menu and proceed to the File Select screen, whereas pressing Select/Start will load a stage based on options set in the menu. Options are stored in SRAM (starting near

$5E02  ) and some (such as the current page) persist after restarting, although others (such as sound test values) are cleared when the menu is opened.

1/7

BG  - When this is enabled, sprite collision points used for collision with the background will be displayed in grey in-game. Pressing L and R buttons at the same time will change their color to black until the pause menu is opened or the map is changed.

OBJ  - When this is enabled, sprite hitboxes used for collision with other sprites will be displayed in-game. The hitboxes seem to disappear entirely when there's no other sprites for Kirby to collide with, maybe to save on CPU time.

RENEW  - When this is enabled, more sprite collision points will be displayed in-game. It's unclear what these are used for. Pressing L and R buttons at the same time will change their color to black until the pause menu is opened or the map is changed.

PAUSE  - When this is enabled, pressing Start will halt the game rather than bring up a separate pause screen. The R button can then be used to advance frame-by-frame. The R button can also be used together with the "up" and "down" buttons on the D-Pad to cycle through all abilities. Pressing R and Select buttons at the same time will open the normal pause screen. In this mode, Gooey is displayed with a small star on his head if he is controlled by the CPU. If Start is pressed on the second controller, a black bar will appear on the screen, and can be removed by pressing Start again on the second controller. It seems to grow bigger depending on the amount of content present on the screen, so it might have been used as a test bar. Holding L and R buttons and pressing a direction on the D-Pad of the first controller will move you to another room, though the game will crash if you try to go to the room to the left of the start of the stage.

LEVEL/STAGE  - Chooses the "world" and stage to load when Select/Start is pressed. B stands for boss. Levels 1 through 5 are self-explanatory, while Levels 6 and T contain miscellaneous maps like mini-games and debug rooms.

6-1 - Final Battle

6-2 - MG5 (mini-game)

6-3 - Boss Butch (boss rush mode)

6-4 - Jumping (mini-game), though no items are present and it softlocks when Kirby leaves the screen

6-5 - Door Test

6-6 - Level 1 opening cutscene, except you spawn in as an extra Kirby with glitched graphics

6-B - Terrain Test

T-1 - Debug Area

T-2 - freezes

T-3 - An odd stage that seems to use screens from 1-4 and 1-5

T-4 - freezes

T-5 - freezes

T-6 - freezes

T-B - freezes

2/7

STOCK  - Chooses the number of lives remaining, offset by one. It ranges from 0 to 99, giving you an actual range of -1 to 98 lives. For some reason, -1 displays as 67 in the life counter, but dying will still result in a Game Over.

KIRBY  - Chooses Kirby's starting health in half-units, from 1 to 10. "0" means 10 (i.e. 5 units of health), and "I" means infinite health.

GOOEY  - Chooses Gooey's starting health in half-units, from 1 to 8. "I" means infinite health.

BGBAR  - Pressing X on the second controller will display a bar to test how elements are placed with the background. Pressing X once will make it appear and you will be able to move it by using the D-Pad on the first controller, pressing X a second time will make it automatically move and pressing X a third time will make it disappear as usual. Pressing L and R buttons at the same time on the second controller will make it disappear while held, and will change the color of the bar and of dust visuals to black (like Kirby running or the dust generated by the Broom ability) until the stage is beaten.

UNIT  - Unknown. When set to SLOW, the game seems to experience more slowdown, most noticeable when a stage begins.

3/7 4/7 5/7

These three screens edit each save file. One thing they can't change is which Heart Stars you've collected.

LEVEL/STAGE  - Chooses the latest level and stage to be unlocked.

OPTION  - These can be set to 0, 1, or 3 with the Y button. When set to 1 or 3, they unlock options in the File Select screen. If 3 is chosen, MG5, Boss Butch and Jumping are considered to be completed, as the overall save file completion percentage also increases.

S - Sound Test

M - MG5

B - Boss Butch

G - Jumping

E - Cutscenes

Note that this is different from the order they actually appear in the File Select screen, with Jumping and Boss Butch switched around.

If the given save file does not exist, it can be created with the Y.  File can also be deleted using Select+X+B; however, Select will also load a stage based on the first screen, so this can only be seen when Select is pressed last (and even then, the stage will be loaded).

6/7 MUSIC ROOM

A sound test. You can play every track and sound effect in the game here. The track stays playing while you navigate the other menus, so enjoy your favorite tune!

Choose the song or sound effect to play with Left and Right, and play it with X, Y, or B. This screen also displays a name for each item, which may have been filenames used in development.

BGM  - Background music.

FGM  - Sound effects, or "foreground music".

7/7 AUTO PLAY DEMO

A screen that seems to be used to record the player's movements for the title screen demos, and then play them back. It doesn't appear to be functional anymore, and tends to delete save data.

Early Boss Room

Entering Acro's boss fight from the debug stage loads into an earlier version of his boss room, similar in appearance to its Boss Butch counterpart, with partially broken Sand Canyon tiles and a unique foreground palette.

Unused Music

K3BOSS2

BGM 3, K3BOSS2 (as named in the  Music Room ), is a fast-paced remix of  Egg Catcher  from

Kirby's Adventure  , situated between K3BOSS1 (main boss theme) and K3BOSS3 (mini-boss theme).

Unused Sound Sample

There's an unused sample which is loaded in Ripple Field 2's sample set, being the first instrument loaded with the sample set at slot 28. It's a pulse wave which is not used anywhere in the song. This could suggest that the song was meant to have a more synth-ish or 8-bit approach at some point, as a 50% square wave is already used in the song, while the unused pulse wave is a 25% square wave.

Oddity

K3EVENT2

BGM 9, K3EVENT2, is only used in the fourth stage of Grass Land (the one with Goku & Chao) in the small room above Boboo rather than the mini-boss theme if the the Heart Star conditions are not met. However, since the preceding track, K3EVENT1, is the "Mission Complete" theme, the title of the track suggests that it was originally the "Mission Failed" theme, which is BGM 29/K3LEVEL5 in the final. Interestingly, the final tracks's name suggests it was once intended to serve as a late-game level map theme.

Also, K3LEVEL3 is used as the map theme for Cloudy Park world while K3LEVEL 4 is used for Sand Canyon, which suggests the levels or maps themes were swapped at some point in development.

Unused Code

In most  Kirby  games, using the Burning ability towards a wall will make Kirby crash and bounce from it, but this game is an exception to this, as Kirby will just awkwardly stick to the wall until the attack ends, however, there's unused code that makes Kirby crash and bounce back just like in other  Kirby  games and will also let him cancel the attack at any moment. This code can be re-enabled by changing ROM address

$04430D

to

9E  . It seems like this was a last minute design choice as this behavior persisted in

Kirby 64  .

(Source:  Pinci )

Unused Tiles

Background Tiles

Description

Background pieces for the Debug stage, while most of them are used in said stage itself, several copies of the moon and star seemingly go entirely unused.

Unused grass tiles resembling those of Ripple Field 1 found in the tile set for Grassland 4, which itself has a surprising lack of grass for a grass stage.

Unused background pieces for the Whispy Woods boss fight, which depict mountains similar to what is seen in his boss fight in

Kirby's Dream Land 2  .

Unused Background pieces for an older non-animated version of the Island background used in Ripple Field 1 and 2.

The tile set used in the bad ending contains tiles for slopes, doors and a block that go unused due to never appearing in the game outside of said cutscene.

Hidden "PERFECT"

There's a hidden screen in the game which is only accessible at the end of Boss Butch mode, and even so, it's rather cryptic to find it. To get to it you have to press X+L+R after defeating Zero, just before the screen fades to black. If done correctly, this screen will appear instead of the credits.

Internal Name

The project's internal name is "K3" as seen at the start of the credits and the Debug menu's sound test.

Regional Differences

Surprisingly, the difference between the International version of  Kirby's Dream Land 3  and the later-released Japanese version comes down to just 41 different bytes. All of the graphics (including regional logos) are found within both versions, and most of the changes merely choose different graphics to be loaded.

The byte that controls which assets to load is in the ROM header's region byte (  0x7FD9  ) - $00 for Japanese, $01 for US. $02-$0C will display the region error on the right (also easily observable by attempting to play the game on a PAL console) and anything else will load the US assets (again).

The game's language can be changed using the following  Pro Action Replay  (PAR) codes:

Japanese to English

English to Japanese

00FFD901

00FFD900

Menus

International

Japan

The Japanese title screen adds an animated squiggle effect to the game's logo; the International version, meanwhile, is static.

International

Japan

Weirdly enough, although the International version features the rōmaji pill-shaped Nintendo logo that has continuously been in use since 1975, the Japanese release features the original Kanji logo used from 1889 to 1950. This is rarely seen in Japanese versions of Nintendo games, with most opting for the usual rōmaji logo.

International

Japan

For some reason, the Japanese version refers to sound effects as FGM (foreground music).

International

Japan

The Japanese version adds a "Nintendo 16" logo, parodying the N64 one, to the Boss Butch title screen, playing off of the SNES being 16-bit compared to the 64-bit N64, which had already been out for nearly two years by the time  Kirby's Dream Land 3  came out in Japan.

Credits

When set to Japanese, the end credits has a unique feature. For the main game's good ending, the credits are displayed in kana, featuring a font that mimics the appearance of a child's handwriting and with a doodle of a red star on each side of the credits' headers. For completing Boss Butch, the credits are listed in English and with a more "computerized" font, which is the case for both versions when in the International setting.

Version Differences

In the SNES version of the game, the background clouds in the Boss Butch rematch against Dark Matter/Zero are blue and red. In the Virtual Console re-release, they are orange and green, most likely out of epilepsy concerns due to their scrolling speed. Additionally, when Zero makes his appearance, the rapid white flashing seen in the SNES version is replaced by slower gray flashes in re-releases, also due to epilepsy concerns.
