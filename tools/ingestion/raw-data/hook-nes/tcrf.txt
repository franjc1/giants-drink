Hidden Text

This game shares the same music driver library that many other NES Ocean Software games used. As a result, there's a hidden version and copyright string hidden at ROM address

0x187C  :

MUSIC DRIVER V1.0 (C)JONATHAN DUNN & OCEAN SOFTWARE

Ending Sequence and Free Map

Press  Right, Up, Up, Down, Down, Right, Left, Up  at the title screen. A sound will confirm correct code entry. Now press  Select  to view the ending sequence or start a new game, select a destination on the map, then press  B  to go there instantly without level completion.

High Scores

US/Europe

Japan

The Japanese version of the game allows for more characters to be input than the international versions, presumably due to the shortened text. As a result, it also reveals the full names of the developers; Mark Rogers (マーク・ロジャース), Chris Kerry (クリス・ケリー), Steve Kerry (スティーブ・ケリー) and Jon Beard (ジョン・ベアード).

Credits

The Japanese version lists the developer credits in katakana (most likely making this the very first video game to have English developers' names in Japanese) and adds a missing credit for Matthew Cannon, the game's sound composer, which was left out of the US and European versions for some reason, most likely because his name couldn't fit on the screen because of the large text.
