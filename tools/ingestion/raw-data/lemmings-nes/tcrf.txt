Unused Graphics

Legal Screen

A graphic that reads "PRESENTED BY IMAGINEER", as a Japanese release by Imagineer was planned at some point (Listed in the HVC Lot Check spreadsheet along with its product code IMA-LE). Note that this is a mockup since the "PRESENTED BY" graphic can be inserted into the "SUNSOFT" graphic.

HUD

The graphics for the HUD include 11 letter characters, which are sufficient for the few words used in it. However, the words used in the HUD only contain 9 of the available 11 letters, so the characters N and R go unused.

Trap Animations

Various lemming animations reminiscent of some traps in the Amiga original that were not kept in the NES version. Note that the graphics for the traps  themselves  are not in the game at all.

Unused Sounds

Sound Effect 02

Sound Effect 07

Sound Effect 0B

Three unused sound effects exist in the game that are never triggered in the game's code. Their intended purposes are unknown.
