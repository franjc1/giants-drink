This game has a  hidden level select .

Level Select

Press  Down, A, Down, Y, Down, A, Down, Y  on the "Start/Options" selection screen. A sound effect confirms the code. During gameplay, pause and press  Up, X, Down, B, Left, Y, Right, A, L, R . When you hear the sound effect confirming the code, you're now able to press  Select  to end any level. Hold  R  while the level's score is being calculated to bring up the level select.

Hidden Level Select Options

On the level select screen, press  L, R, X, B, Left, Up, Right, Down, Y, A  to add extra options for infinite stars, infinite lives, and no enemy collisions. However, for whatever reason, you have to then enter an additional code to be able to toggle each option. The codes are as follows:

Option

Code

Infinite Lives

Up, Down, Up, Down, Left, Right, R, L, Y, A

Infinite Stars

A, Y, L, Up, Down, R, A, Y, Right, Left

No Enemy Collisions

Down, A, Y, R, Y, B, Up, L, Y, A
