The NES

Home Alone  , based on the film of the same name, sees Kevin McCallister try to avoid getting caught by the Wet Bandits for 20 minutes. Yes, 20  real-time  minutes.

Kevin can lay various traps to hinder Harry and Marv, as well as hide in certain places. However, you only get one life, so if you're caught it's game over and the game starts again from the beginning.

Revisional Differences

Secret Screen

At the copyright text screen, press  Up + Left + A + B + Select , then  Down + Right + A + B + Start . This special screen will appear.

This code can be entered with either controller, and can also be done in Bethesda's other NES games (  Wayne Gretzky Hockey

and

Where's Waldo?  ) to bring up a similar screen.

Reset String

Present at

0x06FC  .

RAND

This is a reference to the programmer, Randy Linden. The string checks whether the credits screen has been displayed or not upon reset.

Unused Graphics

Graphics from the title screen of  Where's Waldo?  are present, for whatever reason.

Revisional Differences

Revision 0

Revision 1

Getting caught by Harry or Marv will bring up this screen. Originally, this was art of Kevin doing his famous scream pose. Revision 1 replaced him with a puff of smoke that has "oh no!" in it (make of this what you will), and also moved the picture frames down a bit for some reason.
