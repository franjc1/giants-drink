Debug Menu

To do:  Further documentation.

Jump to Scenario 01
Start Event 01
Press A if you want to look at event
Press B if you don't want to look at event
Press Y to only look at map

At offset

0xFFAD

in a headerless ROM, change the first two values from

FF 00

to

00 FF

(or enter  Pro Action Replay  (PAR) codes

C0FFAD00  ,

C0FFAEFF  ). This will enable a debug menu, accessible by holding  L + R  on controller 2 and resetting the game.

First the scenario select menu (left screenshot) will appear. After selecting a scenario, the debug menu (right screenshot) will appear.

In the debug menu, press  A  to start the chosen scenario as normal,  B  for a battle test, and  Y  for a world map debug screen.

(Source:  byuu , Makainyumon )
