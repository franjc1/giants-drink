Cheats

Enter these codes when paused. If you make a mistake while entering a cheat, press  R  to try again. There's 7 total cheats, two of which were disabled, and one that's unused.

Code

Effect

X, A, Y, Y, B, R

Stage clear.

Y, B, B, A, X, Y, Y, B, R

99 lives.

Y, B, Y, A, X, Y, Y, Y, R

99 hearts

B, A, X, Y, X, A, X, Y, R

Infinite mid-air jumps by pressing  X .

B, B, A, Y, Y, Y, A, B, R

Unused.

X, A, B, Y, Y, Y, X, B, R

Disabled, must add Pro Action Replay (PAR) code

80B14D00

or Game Genie code

DD80-170D  . Use this cheat after collecting a certain number of cheese bits to warp to a stage (see list below).

Disabled, must add Pro Action Replay (PAR) code

80B14D00

or Game Genie code

DD80-170D  . 99 lives, infinite mid-air jumps by pressing  X , 99 hearts, 99 marbles.

Unused Text

Present at hex address

0x8172

is, for whatever reason, some text from  the SNES version of  The Hunt For Red October  .

IN NOVEMBER OF 1984, SHORTLY BEFORE GORBACHEV CAME TO POWER, 
A TYPHOON- CLASS SOVIET SUB SURFACED JUST SOUTH OF THE GRAND BANKS. 
IT THEN SANK IN DEEP WATER, APPARENTLY SUFFERING A RADIATION PROBLEM. 
UNCONFIRMED REPORTS INDICATED SOME OF THE CREW WERE RESCUED. 
BUT ACCORDING TO REPEATED STATEMENTS BY BOTH SOVIET AND AMERICAN GOVERNMENTS, 
NOTHING OF WHAT YOU ARE ABOUT TO SEE... ...EVER HAPPENED.

(Source:  Ragey )

The same text is also present in

Beethoven: The Ultimate Canine Caper  .
