A console port of the arcade wrestling game of the same name, best known for featuring  Mike Haggar  as a playable wrestler. Notably, this and the Genesis port feature redrawn art for the attract sequence and in-game portraits in order to more closely resemble Tetsuo Hara (of

Fist of the North Star

fame)'s promotional art for the series.

Error Messages

Region Error

Attempting to play the American version on a European console throws up the above error message on boot. Like most Capcom games with region lockout-based errors, it will eventually fade out and the game will start as normal. This region lockout check is not present in the Japanese and European versions, and they will load the game normally if loaded on a European or North American/Japanese console, respectively.

Multiplayer Adapter Error

Muscle Bomber

Saturday Night Slam Masters

If the game detects a multiplayer adapter connected to controller port 1, it throws the above error message, complete with Mike Haggar posing and his signature/autograph after the message finishes printing.  Muscle Bomber  includes a header on the error message and is written with a more personalized tone, as opposed to the formal and dry tone of the English message. Hover for translation.

Bad Controller Error

Muscle Bomber

Saturday Night Slam Masters

Same changes made as above. This error appears if the game detects anything but a standard controller or multiplayer adapter connected to any of the controller ports.
