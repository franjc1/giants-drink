Debugging Features

Soft Reset

Pro Action Replay (PAR) code

00833C01

enables a soft reset function: Press  Start + Select  at any time to return to the title screen.

(Source:  Rachel Mae )

Active While Paused

PAR code

00A17A01

enables a few features that are activated while the game is paused:

Select  - Toggles a VRAM viewer. Press  Up  or  Down  to switch pages.

X  - Toggles the HUD area between a set of unknown hex values and the normal HUD.

L  skips to the next stage when the game is unpaused, and  R  restarts the current stage when the game is unpaused.

NOTE:  Using the level skip during the final fight against Ivan Ooze loads a glitchy screen, at which point pressing any button will return to the title screen.

(Source: Rachel Mae)

Active During Gameplay

PAR code

03815201

enables several features that are activated during normal gameplay. Hold  L  and press one of the following buttons to activate the corresponding function:

A  - Increases your power meter by a small amount.

B  - Fully restores your life.

X  - Toggles full invulnerability, though falling in a pit will still kill you.

Y  - Increases your remaining lives by one. Going past 8 will wrap the counter back to 0.

R  - Destroys all enemies on the current screen.

Select  - Cycles through the playable characters.

(Source: Rachel Mae)

Test Mode

To do:  The text for the sound test still exists in the ROM; can it be reactivated?

PAR code

06FBCC01

enables a test menu at the title screen. Highlight OPTION, then hold  L  and press  Start  to activate it. The menu includes the following options:

PLYER TEST  - A controllable character animation test.

SOUND TEST  - Presumably would have allowed various sounds to be played. Unfortunately, in the final game both options' pointers are set to PLYER TEST.

The PLYER TEST controls are as follows:

Up/Y  - Increase values.

Down/B  - Decrease values.

Select  - Switch between "CHR TYPE" (individual characters) and "CHR NO" (individual character frames).

(Source: Rachel Mae, Abystus)

Level Select

To do:  Figure out what that secondary value does, if anything.

PAR code

06FDF601

enables a level select at the title screen: Press  Y/B  to increase/decrease the starting level (right digit). You can also press  X/A  to increase/decrease a secondary value (left digit), though this will freeze the game.

(Source: Rachel Mae)
